# Food Ordering System

##  Project Overview

The **Food Ordering System** is a web-based application that allows users to browse food items, add them to a cart, and place orders online. It provides a smooth and user-friendly experience for customers and an admin panel for managing food items and orders.

---

## Features

###  User Side

*  User Registration & Login
*  Browse food menu
*  Add to cart & manage items
*  Place orders
*  View order history

### Admin Side

*  Add / Update / Delete food items
*  Manage customer orders
*  Manage users
*  Dashboard for monitoring activity

---

## Tech Stack

* 💻 **Frontend:** HTML, CSS, JavaScript
* 🧠 **Backend:** (Flask /php)
* 🗄️ **Database:** MySQL / MongoDB

## ▶️ How to Run the Project

### 🔹 Step 1: Clone Repository

```bash
git clone https://github.com/yourusername/Food-Ordering-System.git
```

### 🔹 Step 2: Navigate to Project

```bash
cd Food-Ordering-System
```

### 🔹 Step 3: Install Dependencies

For Flask:

```bash
pip install -r requirements.txt
```

For .NET:

```bash
dotnet restore
```

### 🔹 Step 4: Run the Application

For Flask:

```bash
python app.py
```
